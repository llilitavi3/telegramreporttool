</p>

</p>

<p>

</p>
<p>
    This tool features a robust multi-account management system and an advanced fail-safe proxy rotation engine. 🚀&nbsp;
</p>
<p>
    Key Features • Multi-Account Management: Login with multiple Telegram accounts and manage them from a single dashboard. • Fail-Safe Proxy Engine: ◦&nbsp;
</p>
<p>
    Supports SOCKS5, SOCKS4, HTTP, and MTProto. ◦ Automatic Distribution: Assigns a unique proxy to every account during reporting. ◦&nbsp;
</p>
<p>
    Auto-Jump &amp; Retry: If a proxy is slow or dead, the app automatically switches to the next IP in the list and retries the report (15s timeout). •&nbsp;
</p>
<p>
    Live Progress Tracking: Watch reports being sent in real-time with status cards and countdown timers between account switches. •&nbsp;
</p>
<p>
    Advanced Target Resolution: High-accuracy extraction of message IDs from public handles, t.me links, and private group URLs (t.me/c/...). •&nbsp;
</p>
<p>
    Session Import: Import existing TDLib session folders via ZIP files—no SMS code required for established accounts. •&nbsp;
</p>
<p>
    Optimized Build: ABI filtering and R8 shrinking result in a lightweight APK (~40MB). 🛠️ Requirements •&nbsp;
</p>
<p>
    Android Studio (Ladybug 2024.2.1 or newer recommended). • JDK 17 or higher. • Android SDK 37 (Compile SDK). •
</p>
<p>
    Real Android Device (arm64-v8a or armeabi-v7a) for testing proxy connections. 📥
</p>
<h3>
    Installation &amp; Setup
</h3>
<p>
    1. Clone the Repository: Shell Script <code>git clone https://github.com/YOUR_USERNAME/telegramreporttool.git</code>&nbsp;
</p>
<p>
    2. Open in Android Studio: ◦ Wait for Gradle to sync. ◦ The app uses a built-in libtdjni.so (TDLib 1.8.40). 3. API Configuration: ◦
</p>
<p>
    &nbsp;The API_ID and API_HASH are currently configured in TelegramManager.kt. It is recommended to use your own from my.telegram.org. 📦
</p>
<h4>
    How to Build Release APK
</h4>
<p>
    To generate the optimized, small-sized APK:&nbsp;
</p>
<p>
    1. Open the Terminal in Android Studio.&nbsp;
</p>
<p>
    2. Run the following command: Shell Script <code>./gradlew assembleRelease</code>&nbsp;
</p>
<p>
    3. The output will be located at: <strong>app/build/outputs/apk/release/app-release.apk</strong> 📂&nbsp;
</p>
